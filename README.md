# publish_article

Java version: 1.8

Memory database: H2

Build tool : Maven

Packaging : WAR

Spring components:
Boot, Data, Rest, Thymeleaf

Front End
Html5, Bootstrap 4, AngularJs

Please, follow the following instructions to build, test & run the application 

Go to project folder,

Build:
$ mvn clean install

Test: 
$ mvn clean test

Run:
$ mvn spring-boot:run

Access App : http://localhost:8080/
